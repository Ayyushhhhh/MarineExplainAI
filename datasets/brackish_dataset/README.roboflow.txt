
Brackish Underwater - v2 960x540
==============================

This dataset was exported via roboflow.ai on October 26, 2020 at 2:57 AM GMT

It includes 14674 images.
Animals are annotated in YOLO v5 PyTorch format.

The following pre-processing was applied to each image:
* Auto-orientation of pixel data (with EXIF-orientation stripping)
* Resize to 960x540 (Stretch)

No image augmentation techniques were applied.
